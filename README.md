# 🌐 Holberton Network

This repository contains networking projects completed as part of the **Holberton School networking curriculum**.

The goal of these projects is to build a strong understanding of **computer networks**, how devices communicate, and how data travels across the internet.

---

# 📡 Project Overview

In this repository I explored the fundamental concepts of networking, including **network protocols, models, and communication mechanisms** used in modern systems.

The exercises focus on understanding how networks function at different layers and how data is transmitted between machines.

---

# 🧠 Topics Covered

## 🧩 OSI Model

Understanding the **OSI (Open Systems Interconnection) model**, which describes how network communication is structured.

Topics include:

* The 7 layers of the OSI model
* Role of each layer in communication
* Data encapsulation
* Interaction between layers

---

## 🔗 TCP Protocol

Study of the **Transmission Control Protocol (TCP)**.

Concepts include:

* Reliable communication
* Connection establishment (3-way handshake)
* Packet ordering
* Error detection and retransmission

---

## ⚡ UDP Protocol

Understanding the **User Datagram Protocol (UDP)** and how it differs from TCP.

Topics include:

* Connectionless communication
* Faster data transmission
* Use cases such as streaming, DNS, and gaming

---

## 🌍 Internet Networking Basics

Additional networking concepts explored:

* IP addressing
* Ports and sockets
* Network communication fundamentals
* Client-server model
* Packet transmission

---
